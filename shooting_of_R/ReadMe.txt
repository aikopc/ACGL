ゲーム名：Rのシューティング	ver.2.46

対象人数：1人
ソフト種別：フリーソフト

この度は「Rのシューティング」をダウンロードしていただきありがとうございます。
本ソフトはramdosが「Hot Soup Processor Script Editor For Windows version3.4」を用いて制作しました。

■　著作権

本ソフトの著作権はRamdosに帰属します。
効果音に魔王魂様、効果音ラボ様のものを使用しました。ありがとうございました。
効果音以外はすべて私が製作しました。
すべて「Source」ファイルに入っています。
使えるクオリティーではと思われますが効果音を除いたすべてのファイルは愛光学園パソコン部の方（とOBの方）は自由に使っていただいてかまいません。
本ソフトの使用により損害が発生しても作者は一切の責任を負いかねます。

■　キー操作

spaceキーでショット
矢印キーで左右移動

これだけです。選択はマウスでどうぞ。

■　ゲームの説明

ただのシューティングです。
ゲージをためてショットを打ち、侵略される前に敵を倒してください。
残機3つで、敵を倒してポイントを得てください。
敵の出現はランダムです。
